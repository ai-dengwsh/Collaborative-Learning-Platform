# 在线协作学习平台 🎓

## 项目简介 📚
这是一个功能强大的在线协作学习平台，旨在为用户提供优质的在线学习和协作体验。

## 主要功能 ✨
- 用户管理 👥
  - 注册登录
  - 个人资料管理
  - 角色权限控制
- 课程管理 📖
  - 课程创建与编辑
  - 课程分类与搜索
  - 课程公告
- 协作功能 🤝
  - 实时聊天
  - 讨论区
  - 协作文档
- 资源管理 📁
  - 文件上传分享
  - 资源权限控制
- 作业系统 ✍️
  - 作业发布与提交
  - 自动评分
  - 截止日期管理
- 数据分析 📊
  - 学习数据统计
  - 参与度分析

## 技术栈 🛠
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：MongoDB
- 实时通信：Socket.IO

## 快速开始 🚀

### 前端启动
```bash
cd client
npm install
npm start
```

### 后端启动
```bash
cd server
npm install
npm run dev
```
